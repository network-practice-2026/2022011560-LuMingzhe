# 计算机网络实习项目

## 项目简介

本项目是一个面向计算机网络课程实习的交互式学习平台，用于展示常见网络协议流程、综合网络访问场景以及计算机网络知识图库。项目通过前端可视化组件和后端知识库接口，将 DNS、TCP、HTTP、ARP 等协议过程以动画、拓扑图和结构化知识的形式呈现，帮助学习者理解协议交互、分层模型和网络设备协作关系。

## 技术栈

- 前端框架：Vue 3
- 构建工具：Vite
- 后端框架：Express
- 数据库：SQLite
- SQLite 驱动：better-sqlite3
- 可视化：ECharts、Mermaid、vis-network
- UI 组件库：Element Plus
- 运行环境：Node.js

## 环境配置

运行项目前需要准备以下环境：

- Node.js 18 或更高版本
- npm 包管理器
- Windows、macOS 或 Linux 终端环境

项目默认端口：

- 前端开发服务：`5173`
- 后端接口服务：`3000`

后端数据库文件位于 `server/knowledge.db`，用于存储知识库、协议数据和仿真数据。

## 安装依赖

在项目根目录执行：

```bash
npm install
```

## 启动方式

### 启动后端服务

在一个终端中执行：

```bash
npm run dev:server
```

后端服务默认运行在：

```text
http://localhost:3000
```

### 启动前端服务

在另一个终端中执行：

```bash
npm run dev
```

前端开发服务默认运行在：

```text
http://localhost:5173
```

### Windows 一键启动

Windows 环境下也可以直接运行：

```bat
start-web.bat
```

该脚本会启动后端服务、打开浏览器并运行前端开发服务。

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 主要功能

### 协议可视化

- DNS 域名解析流程展示
- TCP 三次握手过程展示
- TCP 四次挥手过程展示
- HTTP 请求与响应过程展示
- ARP 地址解析流程展示

### 综合网络场景仿真

- Web 访问场景模拟
- 主机、交换机、路由器、DNS 服务器、Web 服务器之间的交互展示
- ARP、DNS、TCP、HTTP 多协议协同流程演示
- 分步骤播放、回退、重置和跳转控制

### 知识图库

- TCP/IP 五层模型知识展示
- 分层知识条目浏览
- 核心概念、典型协议、相关设备、数据封装和层间协作关系展示
- 知识图谱可视化展示
- 支持创建自定义知识库
- 支持通过 JSON 导入知识库
- 支持为自定义知识库添加知识字段

### 后端数据管理

- 使用 Express 提供协议、仿真和知识库接口
- 使用 SQLite 持久化知识库数据
